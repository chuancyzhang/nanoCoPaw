# Contributing to nanoCoPaw

nanoCoPaw is intentionally small. Contributions must keep it small.

## Scope

- Channel adapters for DingTalk, Feishu, QQ, Discord, iMessage  
- Minimal runtime and binary packaging  

## Non-goals

- Web console, dashboards, or UI  
- Config file systems  
- Skills marketplace, cron, MCP  

## Principles

- Code is configuration: change code, not files  
- Remove unused code instead of disabling it  
- Keep dependency surface minimal  

## Process

1. Fork and open a PR  
2. Describe the change and why it belongs in a minimal core  
3. If tests exist for the modified area, run them  

Thank you for keeping nanoCoPaw sharp and small.
