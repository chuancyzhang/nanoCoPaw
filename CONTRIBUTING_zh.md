# 为 nanoCoPaw 贡献代码

nanoCoPaw 的核心是极小化，请保持它足够小。

## 范围

- 钉钉 / 飞书 / QQ / Discord / iMessage 频道适配  
- 最小化运行时与二进制打包  

## 不做

- Web 控制台或仪表盘  
- 配置文件体系  
- 定时任务、MCP、Skills 市场  

## 原则

- 代码即配置：修改代码而不是写配置  
- 没用的代码直接删除  
- 依赖越少越好  

## 流程

1. Fork 并提交 PR  
2. 说明改动为何符合极简核心  
3. 若有相关测试，请运行  

感谢你让 nanoCoPaw 更小、更清晰。
